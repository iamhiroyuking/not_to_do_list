# Not To Do List

「すべきこと（To Do）」を詰め込むのではなく、「やらないこと（Not To Do）」を決めて生産性を最大化するためのiOS向け習慣化管理アプリケーションです。科学的な行動経済学や心理学の知見を取り入れ、誘惑を断ち切り、目標達成に集中するためのシステムを提供します。

##  開発背景・目的
人間の意志力には限界があり、「〜しない」という否定形の我慢は脳に強いストレスを与え、かえって逆効果になることが知られています。
本アプリは、単なる禁止リストではなく、**「If-Thenプランニング（もし〜したら、代わりに〜する）」**の概念を取り入れ、悪習慣をポジティブな行動へ上書きすることを目的として開発しました。

投資の規律維持、試験勉強への集中、日々のルーティン管理など、ストイックに目標を追いかける人を強力にサポートします。

##  主な機能

### 1. Not To Do リスト & ダッシュボード
- 自分が「やらない」と決めたルールを一元管理。
- 毎日「KEEP（達成）」または「FAIL（未達成）」を1タップでシンプルに記録。

### 2. 習慣化のカレンダー可視化
- 過去の記録をカレンダー上にマッピングし、継続状況を視覚的にトラッキング。
- 連続達成日数をカウントする「Don't break the chain（鎖を断ち切るな）」の心理的ストッパーを実装。

### 3. 失敗パターンの予測と「⚠️危険シグナル」
- 誘惑に負けやすい状況（例：寝不足、焦り、大きな損失の後など）をあらかじめ言語化してメモ。
- 客観的に自分の状態に気づくことで、誘惑を未然に防ぐ防御壁を作ります。

### 4. マインドセット・コラム機能
- 意志力のメカニズムや、行動選択に関する科学的根拠に基づいたTipsをアプリ内で閲覧可能。
- ローカルのHTML/JSONデータを読み込み、滑らかなアニメーションで表示。

### 5. リマインダー（ローカル通知）機能
- ユーザーが自身のライフスタイルに合わせて、毎日の振り返り時間を自由に設定。
- 外部サーバーを介さない端末完結型のローカル通知により、プライバシーに配慮しつつ習慣化を促進。

### 6. システム設定
- 「ダークモード」への対応。
- アプリ内のデータをワンタップで安全に一括削除できるデータリセット機能。

## 使用技術（Technical Stack）

- **Language**: Swift 5.10+
- **Framework**: SwiftUI
- **Data Persistence**: SwiftData
- **Notification**: UserNotifications (Local Notifications)
- **UI Component**: WebKit (`UIViewRepresentable` によるHTMLコラム表示)
- **Architecture**: MVVMパターンのエッセンスを取り入れたモダンなコンポーネント指向

## 📐 設計のこだわり
- **データ管理の最適化**: `SwiftData` を採用し、リレーショナルなデータ構造（アイテムと日々の記録）を軽量かつ安全に永続化しています。
- **無駄のないUI/UX**: ユーザーに余計なノイズを与えないよう、モノトーンを基調としたミニマルで洗練されたデザインを採用。スプラッシュ画面からコア機能まで統一感を持たせました。

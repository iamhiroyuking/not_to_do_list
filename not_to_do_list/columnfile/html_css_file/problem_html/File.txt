        <p>「睡眠時間を削って努力する」ことは、もはや美徳ではありません。それは、エンジンのオイルが切れた車で無理やり高速道路を走らせるような、極めて効率の悪い、自虐的な行為です。</p>
        
        <h2>飲酒状態と同じ判断力</h2>
        <p>睡眠不足の脳が受けるダメージは、私たちが想像するよりもはるかに深刻です。17時間から19時間連続で起きている人の認知能力は、<span class="highlight">血中アルコール濃度0.05%（酒気帯び状態）と同レベル</span>まで低下するという研究結果があります。この状態で難しいロジックを組んだり、相場の動向を判断したりすることは、泥酔して運転するのと同じくらいリスクの高い行為です。</p>
        
        <h2>睡眠中に「学習」が完了する</h2>
        <p>睡眠は単なる休息ではなく、脳の「メンテナンス時間」です。日中に学習したSwiftの文法やIELTSの単語は、寝ている間に脳内の海馬から大脳皮質へと転送され、長期記憶として固定されます。つまり、<span class="highlight">睡眠を削って勉強時間を増やすことは、覚えたことを脳に刻むプロセスを自ら破壊している</span>ことに他なりません。</p>
        <p>また、睡眠不足は感情を司る「扁桃体」を過敏にし、理性を司る「前頭前野」との連携を弱めます。これにより、普段なら冷静に流せるストレスに過剰反応したり、投資でパニックに陥ったりする確率が飛躍的に高まります。</p>
        
        <h2>「寝ないこと」をリストから除外する</h2>
        <p>「やらないことリスト」の最も重要な項目の一つは、夜更かしを助長する習慣の排除です。「深夜12時を過ぎての作業はしない」「寝る1時間前にスマホを見ない」。睡眠を聖域として守ることで、あなたの脳は翌日、最高のパフォーマンスを発揮できるようになります。最短距離で目標に到達したいのなら、まずは枕に頭を預ける勇気を持ってください。</p>

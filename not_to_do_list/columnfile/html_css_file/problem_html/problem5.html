偽りの報酬

脳は時として、あなたを幸せにするためではなく、ただ「行動させるため」にあなたを誘惑します。これが、悪習慣がもたらす「偽りの報酬」の正体です。

ドーパミンは「快楽」ではない

最新の脳科学において、ドーパミンは快楽を感じる物質ではなく、「期待」を抱かせる物質であることがわかっています。「スマホを見れば面白い情報があるかも」「お菓子を食べれば満足できるかも」という期待を抱かせ、あなたを突き動かします。しかし、実際にその行動をとった後に得られるのは、期待したほどの快楽ではなく、むしろ虚脱感や自己嫌悪である場合が少なくありません。

報酬系のハイジャック

投資の短期的な値動きをチェックしたり、SNSの通知を追いかけたりする時、脳内ではドーパミンが大量に放出されています。これは、原始時代の人間が獲物を見つけた時の反応と同じです。現代のデジタルデバイスや刺激の強い食品は、この脳の「報酬系」を巧妙にハイジャックし、本来あなたが向かうべき長期的な目標（IELTSの学習やスキルの習得）から目を逸らさせてしまいます。

「脳の嘘」を見破る

誘惑に襲われた時、それは「あなたがそれを欲している」のではなく、単に「脳がドーパミンによって期待させられているだけ」だと考えてみてください。このメカニズムを知ることで、衝動を一歩引いた視点で観察できるようになります。次にスマホに手が伸びそうになったら、自分に問いかけてみてください。「これは本物の報酬か、それとも脳が見せている偽りの報酬か？」と。

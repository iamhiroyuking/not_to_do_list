マルチタスクの幻想

「リスニング音声を流しながら別の作業をする」「コードを書きながらSNSの通知をチェックする」。一見、時間を有効活用しているように思えるこれらの行動こそが、あなたの脳のパフォーマンスを劇的に下げている真犯人です。

脳は「並列処理」ができない

コンピュータのマルチコアプロセッサとは異なり、人間の脳（特に意識的な思考を司る前頭前野）は、複数の高度なタスクを同時に並列処理するようには設計されていません。心理学の研究によれば、私たちが「マルチタスク」だと思っているものの正体は、実は2つのタスク間を高速で行き来する「タスクスイッチング」に過ぎません。

この切り替え作業は、脳にとって非常に大きな「スイッチング・コスト」を強います。タスクを切り替えるたびに、脳はそれまで行っていた作業の記憶を一時保存し、新しい作業に必要な情報をロードし直す必要があるため、エネルギーを無駄に浪費し、IQが一時的に10ポイント近く低下するというデータも存在します。

「15分」の空白時間が奪われる

さらに恐ろしい事実は、一度途切れた集中力が元の深さに戻るまでにかかる時間です。カリフォルニア大学の研究では、作業中にメールや通知によって注意が逸れた場合、元の深い集中状態（フロー状態）に戻るまでに平均して約23分15秒かかることが示されています。

「たった数秒スマホを見ただけ」のつもりが、実際にはその後の15分以上のパフォーマンスを犠牲にしているのです。IELTSの難解な長文読解や、Swiftでの複雑なロジック構築において、この数秒の妥協は致命的な学習効率の低下を招きます。

シングルタスクこそが最強の戦略

圧倒的な成果を出す人々は、複数のことを器用にこなすのではなく、一つのことに完全に没入する技術に長けています。だからこそ、「やらないことリスト」によって環境を整える必要があります。作業中に不要なブラウザのタブを閉じること、スマホを物理的に遠ざけること。これらによって「注意の残余（Attention Residue）」を防ぎ、あなたの全エネルギーを目の前の一行のコード、一つの英単語に注ぎ込みましょう。

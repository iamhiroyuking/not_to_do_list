危険シグナルを予測する

習慣が破られてしまう時、そこには必ず「引き金（トリガー）」となる状況が存在します。

失敗のパターンを知る

例えば、「感情的なトレードをしてしまう」のはどんな時でしょうか？大きな損失を出した後かもしれませんし、寝不足でイライラしている時かもしれません。

自分がどんな条件の時に誘惑に負けやすくなるのか、そのパターンを事前に予測しておくことが非常に重要です。

気づくだけで強力な防御になる

アプリの「⚠️危険シグナル」の項目に、あなたの失敗パターンをメモしておきましょう。

「あ、今まさにメモした『焦っている状態』だ」と客観的に気づくことができれば、誘惑の波に飲み込まれる前にスッと身を引くことができるようになります。

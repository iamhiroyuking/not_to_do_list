「やらない」を決める

アップルの創業者スティーブ・ジョブズは、「集中するとは、集中すべきことに『イエス』と言うことではない。他の何百もの良いアイデアに『ノー』と言うことだ」と語りました。

捨てることから始まる

新しい目標（勉強、筋トレ、読書など）を立てる時、私たちはつい「やるべきことリスト」だけを増やしてしまいます。しかし、1日の時間と体力は有限です。何かを新しく始めるには、今やっている何かを辞めなければなりません。

あなたのボトルネックは？

あなたの目標達成を一番邪魔している「無意識の習慣」は何でしょうか？

ダラダラとスマホを見る時間、つい感情的になってしまう瞬間、夜ふかし…。まずはそれらを見つけ出し、「やらないことリスト」に追加して、本当に大切なことのための「余白」を作りましょう。

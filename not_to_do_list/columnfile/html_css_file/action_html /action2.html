代わりの行動を用意する

「絶対にピンク色の象を想像しないでください」と言われると、どうしても頭に思い浮かべてしまいますよね。実は、人間の脳は「〜しない」という否定形をうまく理解できません。

我慢は逆効果になる

「お菓子を食べない」「スマホを見ない」と念じれば念じるほど、脳はかえってその対象を強く意識してしまいます。

これを防ぐためには、単に禁止するのではなく「代わりに何をするか」という新しい行動で上書きするのが効果的です。

「If-Then」のルール

「もし（If）スマホを触りたくなったら、その代わり（Then）本を1ページ開く」「もしイライラしたら、深呼吸を3回する」といった具合です。

「やらないことリスト」が辛い我慢大会にならないよう、代わりのポジティブな行動を自分の中でセットにしておきましょう。

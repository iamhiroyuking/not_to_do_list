記録して可視化する

人間の記憶は驚くほど曖昧です。3日前の夜に自分が何分スマホを見ていたか、正確に答えられる人はほとんどいません。

事実はデータにしか宿らない

「最近だらけているな」というネガティブな思い込みや、「今回はうまくやれている」という楽観的な錯覚を防ぐには、毎日の行動を客観的に「記録」するしかありません。

鎖を断ち切らない（Don't break the chain）

毎日アプリで「KEEP」を記録し、カレンダーにマークが埋まっていくのを見ることは、脳にとって大きな報酬になります。

「ここでサボったら、せっかく続いている10日間の記録がゼロになってしまう！」という心理的な抵抗感が、最後の最後であなたを誘惑から引き留める最強のストッパーになってくれます。

完璧じゃなくても戻る

目標を達成する人とそうでない人の違いは、「一度も失敗しないこと」ではありません。「失敗したあとに、どれだけ早く軌道修正できるか」の差です。

「2日連続」で破らないルール

飲み会や体調不良、あるいは意志力が切れてリストの約束を破ってしまう日は、誰にでも必ずやってきます。

そんな時は、無理に完璧を取り戻そうとせず「2日連続でサボることだけは絶対に避ける」というルールを持ってみてください。

FAILは成長のチャンス

破ってしまった日は、アプリに「FAIL（失敗）」の記録と、その時の理由をメモしましょう。「なぜ負けたのか」を記録するその行為自体が、すでに立派な自己管理の再スタートになっています。
